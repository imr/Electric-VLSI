Gerber is a printed-circuit board layout format, originally from Gerber Scientific. For more information on reading Gerber, see Section 3-9-2.

Gerber options are controlled with the Gerber Preferences (in menu File / Preferences..., "I/O" section, "Gerber" tab).

This dialog has two controls:

  • "Fill polygons" requests that polygons be filled-in instead of outlined.
  • "Read all .GBR files in the directory" requests that the import function scan for other files ending in ".GBR" and read all of them.
